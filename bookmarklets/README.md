# bookmarklets/ — Bauhaus Retur-Webb bokmärken

Dessa bokmärken kopplar interna Bauhaus-system (Puzzel, Magento, DHL Active Tracing,
Bauhaus Logistics) till bauhaus-retur-web. Sparade i repot 2026-07-06 för
versionshantering — tidigare fanns de bara i Adams webbläsares bokmärkesfält.

Installation: skapa ett bokmärke i webbläsaren, klistra in koden nedan som URL.
Filerna nedan är rena referenskopior — redigera INTE dessa .js-filer och
förvänta dig att bokmärket i webbläsaren uppdateras automatiskt; bokmärket
måste uppdateras manuellt i webbläsaren om koden ändras här.
